# Organizador

This is a small Ruby on Rails project, made to scheduled tasks, assign responsible or follower users, ad fies to a task. Try it!

## Technologies

- Rails 7.0.4
- Ruby 3.1.2
- Bootstrap 5.0
- Postgresql
- Turbo Drive
- CSS
